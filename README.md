# digital garden

my digital garden — notes growing over time. incomplete, evolving, alive.

## what is this

a digital garden is unlike a blog. notes aren't timestamped posts that scroll into the archive. they get revised, cross-linked, abandoned, and revived. some things bloom, some die back, and the whole thing changes shape across seasons.

## adding a new note

1. copy `notes/TEMPLATE.html`
2. replace `TITLE`, `YYYY-MM-DD`, and `STATUS` (e.g., "seedling", "evergreen", "wilting")
3. write raw HTML in the `<article class="note-content">` block
4. link it from `index.html` under the `<ul class="notes">`

## status tags

- **seedling** — just planted, rough, incomplete
- **growing** — taking shape, actively tended
- **evergreen** — stable, regularly updated
- **wilting** — no longer maintained, kept for reference
